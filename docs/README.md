## Deployment Notes

### Custom Domain Setup
This site is configured for the custom domain `shadowhawk.cloud`:
1. Ensure all files are in the `docs/` directory
2. Configure DNS settings to point to hosting provider
3. Set up SSL/TLS certificate for HTTPS
4. Update GitHub Pages settings if using GitHub hosting

### Post-Deployment SEO Tasks
1. Submit sitemap to Google Search Console (https://shadowhawk.cloud/sitemap.xml)
2. Submit sitemap to Bing Webmaster Tools  
3. Monitor for indexing issues
4. Set up Google Analytics tracking
5. Test structured data with Google's Rich Results Test
6. Monitor Core Web Vitals performance
